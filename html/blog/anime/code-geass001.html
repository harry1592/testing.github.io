反叛的魯路修 Code Geass

出品年代： 2006年

類型: 悲劇、科幻、戰爭、機器人動畫

原作: 大河內一樓、谷口悟朗

動畫製作: 日本日昇

故事簡介

皇曆2010年8月10日，超級大國「不列顛尼亞帝國」為了奪取日本的地下資源而侵略日本，在不列顛尼亞帝國強大的兵器Knightmare進攻下，日本在不到一個月的時間內被征服。日本被剝奪了自由、權利甚至是國家的名字，不列顛尼亞帝國將「日本」改稱為「11區」，「日本人」這名詞則被11這個數字所取代。表面上看來不列顛尼亞帝國的統治似乎牢不可破，但其中早已出現了裂痕。 七年後的2017年，在生死邊緣獲得C.C.授予「Geass」的力量，立志要粉碎不列顛尼亞帝國的黑色皇子魯路修，以及獲得第七世代人型機甲蘭斯洛特，立志從帝國內部進行改革的白色騎士樞木朱雀，影響了整個世界。(來自維基百科)

相信有追過老番的朋友，都一定聽過反叛的魯魯修這部經典神品。一部稱得上神作的作品，往往離不開優秀的劇本和突出的人物塑造，而反叛的魯魯修就將這兩個要素做到近乎完美。


魯魯修的劇情充滿懸疑和智鬥,你永遠猜不透下一秒會發生什麼,這種神秘感加上出神入化的運鏡,讓人忍不住一集接一集地住下看。除此之外，劇中還包含了機戰、校園戀愛、超能力、打鬥、心計、神展開等元素,使這一部作品的內容更多元化和大眾化,令整部作品更上一層樓。


劇中的人物每一個都形象鮮明且貼近現實。無論是失去兄長,要把所有包袱背負在自己身上,過著雙面人生的紅月卡蓮

還是殺死父親試圖終結戰爭,卻終其一生受到良心責備的樞木朱雀

不老不死但內心十分脆弱的C.C.

出生於帝王之家,天真卻毫不任性的尤菲美亞

劇中的每一個角色都擁有著自己的掙扎以及迷惘,正是這一點導致許多時配角的魅力甚至能蓋過男主角,尤其是羅洛。

這個角色在前期絕對不是一個令人喜歡的角色,但製作組在最後僅用了五分鐘的劇情配合那傷感的BGM和獨白,完全扭轉了觀眾對他的印象,而他的結局也是令人遺憾。

本劇最大的看點是看著主角魯魯修與朱雀一步步成長,這兩個從一開始便注定走上不同道路的摯友,用著屬於自己的方法來改變這個世界。

不論是當初只想復仇到後來甘願犧牲自己,為人們帶來明天的魯魯修。

還是起初空有一腔熱血卻不知如何執行,到後來帶起面具背負一切的朱雀。

大河內(作者)為我們展示了角色如何在面對不怎麼溫柔的世界時漸漸的成長。

(以下內容涉及劇透，未看過的觀眾請回避)

深層含義

然而這部看似完美的作品也存在一些爭議,戰爭的起源其實是來自於人性的貪婪,貪婪是人類的本性之一,要終止貪婪根本就是不可能的事,所以零之鎮魂曲是違反人性,而且過於理想化,注定失敗。魯魯修只是白白的犧牲了自己的生命,並且為自己的罪行而贖罪而已。

既然零之鎮魂曲必然失敗,那魯路修的犧牲就是不是毫無意義呢?在他們的世界,確實是,但作為一套動漫,零之鎮魂曲確實影響了許多觀眾。或許這首鎮魂曲一開始針對的聽眾就不是劇中的人物,而是現實中觀看這套動畫的觀眾。

劇中曾經出現過三種完全不同的哲學思想:


渴望過去的皇帝夏魯魯

希望把時間定格在今天的修奈傑爾

以及追求明天的魯路修

即使經過尤菲,夏莉以及羅洛的死,魯路修並沒有選擇埋藏在過去並停濟不前,即使背後已經發生過無數的悲劇,魯魯修依然決定把一切背負在身上並且依然對人類以及未來懷抱著希望,而這種精神才是導演想要表達的。

劇場版

在復活的魯魯修劇場版中,許多角色或多或少都覺得虧欠魯路修,他們抱著遺憾在魯路修創造的和平世代生活,原本這些是他們所要背負一生的後悔,但是這次劇場版給予這些角色們救贖,這或許也是作者對於魯路修這部作品最後的溫柔了。


不過最後能看到魯魯修和C.C修成正果，也算是圓了觀眾的這麼多年的心願。

推薦觀看次序

TV版R1>R2>劇場版三部曲(可看可不看對主線影響不大)>復活的魯魯修>外傳亡國的阿基德(故事發生在TV版R1 R2之間的歐洲戰場,有新主角出現)

魯魯修的作畫水平簡直堪稱頂尖,即使是十多年後的今天依然能超越一堆新番,配上當年精良的製作以及恰到好處的配樂,很難想像這是一部15年前的動漫,這絕對是一部值得反覆觀看的佳作!
